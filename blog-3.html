Now we're going to get the actual tweets in a csv format. Twitter allows for over 3000 of the most recent tweets from an account to be downloaded.

def get_all_tweets(screen_name):

    auth = tweepy.OAuthHandler(consumer_key, consumer_secret)
    auth.set_access_token(access_key, access_secret)
    api = tweepy.API(auth)

        #initialize a list to hold all the tweepy Tweets
    alltweets = []

    #make initial request for most recent tweets (200 is the maximum allowed count)
    new_tweets = api.user_timeline(screen_name = screen_name,count=200)

    #save most recent tweets
    alltweets.extend(new_tweets)

    #save the id of the oldest tweet less one
    oldest = alltweets[-1].id - 1

    #keep grabbing tweets until there are no tweets left to grab
    while len(new_tweets) > 0:


        #all subsiquent requests use the max_id param to prevent duplicates
        new_tweets = api.user_timeline(screen_name = screen_name,count=200,max_id=oldest)

        #save most recent tweets
        alltweets.extend(new_tweets)

        #update the id of the oldest tweet less one
        oldest = alltweets[-1].id - 1



    #transform the tweepy tweets into a 2D array that will populate the csv 
    outtweets = [[tweet.id_str, tweet.created_at, tweet.text.encode("utf-8")] for tweet in alltweets]

    #write the csv  
    with open('%s_tweets.csv' % screen_name, 'w') as f:
        writer = csv.writer(f)
        writer.writerow(["id","created_at","text"])
        writer.writerows(outtweets)

Let's read the csv in and start on cleaning the data. 

import pandas as pd
tweets = pd.read_csv('my_tweets.csv')

Removing the retweets that don't have any text from the author before them 

new_tweets = tweets[~tweets.text.str.contains("b\'https:")]

new_tweets = tweets[~tweets.text.str.contains("b\'RT")]

new_tweets = new_tweets[~new_tweets.text.str.contains("b\"RT")]

Dropping the fist two columns because only the text is useful, and remove the b from the beginning of each. 

new_tweets=new_tweets.drop('id', axis=1)
new_tweets=new_tweets.drop('created_at', axis=1)
new_tweets['text'] = new_tweets['text'].str[1:]

Now to remove the url shortcuts (images, links) from the tweets

import re

new_tweets['text'] = new_tweets['text'].apply(lambda x: re.split('(\\\\)\w+', str(x))[0])
new_tweets['text'] = new_tweets['text'].apply(lambda x: re.split('https:\/\/.*', str(x))[0])

Removing the codes for emojis, tab spaces etc

new_tweets['text'] = new_tweets['text'].apply(lambda x: x.replace('\\', " "))

The previous has emptied out some of the rows that only contained a link or an emoji

empty = new_tweets.index[new_tweets.text=="'"].to_list()
new_tweets = new_tweets.drop(empty)

Lastly, we're going to reset the index and and drop the separate one. 

new_tweets.reset_index(inplace=True)

new_tweets.drop('index', axis=1)

Now the tweets are now just the text content in a single column dataframe. You can inspect each element using iloc, and format them for whatever purpose you need them for.
